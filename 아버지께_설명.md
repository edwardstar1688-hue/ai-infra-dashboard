# 아버지께 보여드릴 설명

이 파일은 기존 Python 분석 코드를 웹 대시보드 형태로 바꾼 버전입니다.

## 목표

아버지께서 매번 Python 파일을 직접 실행하지 않고,
웹사이트 링크를 열면 최신 상대강도 차트를 바로 확인할 수 있게 만드는 것입니다.

## 기본 구조

기본 비교 대상은 다음과 같습니다.

AI 인프라:
- SOXX
- QQQ

플랫폼 바스켓:
- GOOGL
- MSFT
- AMZN
- META

## 대시보드에서 확인 가능한 내용

- SOXX / 플랫폼 바스켓 상대강도
- QQQ / 플랫폼 바스켓 상대강도
- 최근 1주, 1개월, 3개월 변화율
- 현재 흐름 해석
- CSV 다운로드

## 아버지 사용 방식

배포가 완료되면 아버지는 아래처럼 사용하시면 됩니다.

1. 웹 링크 접속
2. 기간 선택
3. 비교할 종목 선택
4. 그래프와 해석 확인

Python 설치나 명령어 입력 없이 사용할 수 있습니다.

## 주의점

데이터는 yfinance를 통해 Yahoo Finance 데이터를 가져옵니다.
무료 데이터 기반이므로, 전문 투자 시스템처럼 100% 실시간/기관급 데이터는 아닙니다.
하지만 시장 흐름을 확인하는 대시보드 용도로는 충분히 사용할 수 있습니다.
